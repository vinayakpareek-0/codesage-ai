# CodeSage AI

> Your AI-powered code review companion built on AWS serverless architecture

An intelligent code analysis system that leverages AWS Bedrock's Claude 3 Sonnet to provide instant, comprehensive code reviews. Submit your code, get AI-powered insights on security vulnerabilities, performance bottlenecks, code quality issues, and best practices, all through a simple REST API.

## Tech Stack

- **AWS Lambda** - Serverless compute for scalable code execution
- **AWS Bedrock** - Claude 3 Sonnet for AI-powered code analysis
- **AWS S3** - Object storage for code submissions and results
- **AWS API Gateway** - RESTful API endpoints
- **Python 3.11** - Lambda runtime and application logic
- **boto3** - AWS SDK for Python
